# AI-Powered Customer Support & Service Automation

**Internship Program:** Nasscom SmartBridge - Salesforce Administrator with AI Agentforce
**Organization:** Ganpat University
**Team:** Itisha Desai (Lead) & Nitish Marker

## 📌 Project Overview
This project replaces manual support documentation with a secure, automated Salesforce platform. It leverages **Agentforce** and **Generative AI** to automate record summaries, case routing, and service workflows, significantly reducing administrative effort.

## 🛠️ Key Technical Features
* **AI Integration**: Custom prompt templates built with **Prompt Builder** and grounded in record data.
* **Service Automation**: Record-Triggered Flows for intelligent case classification and queue routing.
* **Data Integrity**: Custom validation rules to ensure high-quality data for AI processing.
* **Custom Schema**: Implementation of objects for Customers, Merchandise, Orders, and User Cases.
